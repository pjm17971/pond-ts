# Alignment and Aggregation

## Aligning onto a grid

Use `align(...)` to sample a series on a sequence grid and produce an interval-keyed view.

```ts
import { Sequence } from 'pond-ts';

const aligned = series.align(Sequence.every('1m'), {
  method: 'hold',
});
```

### Alignment methods

- `hold`: carry forward the latest known value
- `linear`: interpolate numeric values for time-keyed series

### Sampling

Sequence sampling supports:

- `begin`
- `center`

## Aggregating into buckets

Use `aggregate(...)` to reduce source events into interval buckets.

```ts
const aggregated = series.aggregate(Sequence.every('5m'), {
  cpu: 'avg',
  requests: 'sum',
  host: 'last',
});

const renamed = series.aggregate(Sequence.every('5m'), {
  cpu_avg: { from: 'cpu', using: 'avg' },
});
```

Supported built-in reducers:

- `sum`
- `avg`
- `min`
- `max`
- `count`
- `first`
- `last`

You can also pass custom reducer functions in the mapping. See the
[Aggregation Playbook](./aggregation-playbook) for reducer contracts and examples.

## Bucket semantics

Sequence bucketing uses half-open intervals:

- `[begin, end)`

That means a point event at the exact bucket end belongs to the next bucket, not the current one.
