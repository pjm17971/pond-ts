# @pond-ts/react

React hooks for pond-ts live time series: `useTimeSeries`,
`useLiveSeries`, `useSnapshot`, `useLiveQuery`, `useDerived`, `useWindow`,
`useCurrent`, `useLatest`.

:::note
Guided per-hook prose is in progress — until those pages land, the
**[generated API reference](/api)** covers every hook with full
signatures and type information. The `@pond-ts/react` package reference
is split out separately from the core `pond-ts` reference at the same
entry point.
:::

**Install** — `npm install pond-ts @pond-ts/react`. The peer dependency
is React 18 or 19.

**Mental model.** `pond-ts/react` is a thin, render-safe subscription
layer over the live-series primitives. Hooks fall into three groups:

- **Subscription hooks** — `useLiveSeries`, `useTimeSeries` own the
  lifecycle of a live source.
- **Snapshot hooks** — `useSnapshot`, `useLiveQuery`, `useDerived`,
  `useWindow` read a batch `TimeSeries` out of a live source for
  rendering.
- **Reducer hooks** — `useCurrent`, `useLatest` return a single current
  value with per-field reference stability so downstream `useMemo`
  doesn't invalidate unnecessarily.
